Exporting Data

WeatherLog can export the data entered, for if you want to have a backup of the data or share it with someone else. To export data, either click the save button on the toolbar, select Export from the Weather menu, or press Control-S. Then enter the filename to use for the exported file.

However, this is only really useful if the only application you want to use this data with is WeatherLog. That's why there are two more options, for exporting data to HTML or CSV. Export to HTML, available from the Weather -> Export to menu, formats the data into a table that can be viewed in a web browser. Export to CSV, in the same menu, formats the data as CSV (comma-separated values) for use in a spreadsheet program.

If you want to easily share the data with others, you can also export the data to Pastebin, as either raw data or formatted as HTML or CSV.

Note that the export feature only works with the data stored in the current profile, not all of them.
