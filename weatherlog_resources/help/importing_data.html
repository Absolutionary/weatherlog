Importing Data

The exported data is useless if you can't do anything with it! That's why there's also an import feature, to put the data back into WeatherLog. Importing data is just as simple as exporting it; click the open button in the toolbar, select Import from the File menu, or press Control-O. This will remove all of the data in the current profile and replace it with the imported data.

There's another option for importing the data; if you don't want the new data to replace the data in the current profile, select Import and Merge from the File menu. This differs from the other import feature in that it combines both sets of data. If there are no conflicting dates, where a date appears in both the original and new sets of data, then all of the data will be combined into one. However, if there is a date that appears in both sets, then only the data corrosponding to that date in the original set will be added.

Import as New Profile, in the same menu as the other two, is slightly different; it will import data in a new profile, instead of in the current one.
